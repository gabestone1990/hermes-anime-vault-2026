## Overview

Hermes Stream Vault is meant for users who want a tidy way to locate anime episodes, keep track of what they plan to watch, and preserve local copies through a browser-based desktop interface. Search, selection, tracking, and download handling all live in the same place, so you can go from discovery to action without switching between separate utilities.

At its core, the project focuses on anime metadata and streaming source handling that can adapt to different library sizes and workflows. The combination of multilingual metadata support, fuzzy search, and offline local cache features makes it practical for people managing long-running series, broader collections, or repeatable batch and selective download routines.
